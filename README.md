# 🎨 FgG ArtAssist - イラスト制作支援ツール (β版)

**FgG ArtAssist** は、Stable Diffusion WebUI (Automatic1111) のAPIを利用して、イラスト制作のワークフローを効率化するためのデスクトップアプリケーションです。
Gradioを用いたシンプルなUIで、ラフ画の清書や、顔・手の修正を直感的に行うことができます。

## ✨ 主な機能

1.  **ラフ画チェック**
    * 描いたラフ画を読み込み、AIによる清書・修正案を生成します。
    * プロンプトは日本語で入力可能（自動で英語に翻訳され、品質タグが付与されます）。
    * 顔や手の崩れを抑えるための **ADetailer** 設定をプルダウンで簡単に指定可能です。

2.  **顔チェック**
    * キャラクターの顔修正に特化したモードです。
    * ADetailerを利用して、指定した表情や特徴に合わせて顔部分のみを高画質化・修正します。

3.  **SDサーバー管理機能**
    * アプリ内からローカルの Stable Diffusion WebUI を APIモードで起動できます。
    * サーバーの起動状態やログをアプリ内で確認可能です。

4.  **比較・履歴機能**
    * 元画像と生成画像をワンクリックで切り替えて比較できます。
    * 生成された画像のパラメータ（PNG Info）を確認できます。

## ⚙️ 動作環境・必須要件

このツールを使用するには、以下の環境が必要です。

* **Python 3.x**
* **Stable Diffusion WebUI (Automatic1111)**
    * ローカル環境にインストールされていること。
    * **必須拡張機能**: [ADetailer](https://github.com/Bing-su/adetailer) がWebUIにインストールされていること。
* **必須モデル (Checkpoint)**
    * コード内の設定により、推奨モデルとして **`waiNSFWIllustrious`** を含むモデル名が検索されます。
    * *※このモデルがない場合、エラーが発生したり、自動でダウンロードを促すメッセージが表示される場合があります。*
